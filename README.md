# 高校毕业生就业管理系统

## 项目简介
基于Java的高校毕业生就业管理系统是基于Java语言，SSM框架，JSP技术，Mysql数据库的管理高校毕业生就业信息的管理系统。系统支持三个角色：1.管理员 2.学生 3.用人单位 。功能上：1.用人功能管理 2.就业信息管理 3.岗位新管理 4.招聘信息管理 5.就业派遣管理 6.个人中心等。适合作为：毕设设计 、大作业等，资料中包括完整的万字毕业论文。
